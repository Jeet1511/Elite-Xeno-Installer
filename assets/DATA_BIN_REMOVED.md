# data.bin removed from repository

`data.bin` is intentionally **not published** in this repository.

## Why it was removed
The payload can be extracted/decrypted by unauthorized people if shared publicly.
To protect project security and prevent misuse, this file is excluded from source control.

## For maintainers
- Keep `data.bin` private.
- Never commit payload binaries to public repositories.
- Generate and distribute payloads through secure private channels only.
